# Requisitos: Estante Pessoal

## Objetivo

Criar um frontend separado para um sistema de organizacao de livros pessoais, autores, categorias e emprestimos.

## Requisitos funcionais

- Exibir um dashboard com resumo da estante.
- Permitir CRUD de livros.
- Permitir CRUD de autores.
- Permitir CRUD de categorias.
- Permitir CRUD de emprestimos.
- Permitir marcar emprestimos como devolvidos.
- Exibir mensagens por toast.

## Requisitos tecnicos

- Usar apenas HTML, CSS e JavaScript puro.
- Usar ES Modules com `type="module"`, `import` e `export`.
- Nao usar JavaScript inline no HTML.
- Nao usar `alert()` como mensagem principal.
- Manter Tabler local no projeto.
- Manter dados mockados em memoria nos services.
- Preparar `httpClient.js` para futura integracao REST com backend Java.
